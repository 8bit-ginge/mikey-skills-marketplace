# Changelog

All notable changes to the Skill Marketplace Manager plugin.

## v0.2.0 — 2026-04-02

Natural language skill detection and implementation.

## [0.1.0] - 2026-04-01

### Added

- `/marketplace:validate` — validates skills and plugins against Anthropic spec and project conventions
- `/marketplace:new-skill` — scaffolds new skill containers with correct structure
- `/marketplace:new-plugin` — scaffolds new plugin containers with correct structure
- `/marketplace:status` — shows sync state between local artifacts and the marketplace repo
- `/marketplace:publish` — publishes skills and plugins to the GitHub marketplace
- `/marketplace:marketplace-manager` — natural language interface for all commands
